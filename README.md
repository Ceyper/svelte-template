# Ceyper svelte template

A template for creating svelte projects.

## Installing

```bash
mkdir my-project
cd my-project
npx degit ceyper/svelte-template .
git init
npm install
```

## Running

```bash
npm run dev
```

## Building

```bash
npm run build
npm run preview # to preview production build
```

## Formatting

```bash
npm run format # formats project using prettier
```
